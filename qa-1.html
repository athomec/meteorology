常見問題

VMware Horizon Client 疑難雜症 Q&A

  1. 目前建議使用之 VMware Horizon Client 版本為 8，尚未安裝或更新者，請參考 下載 VMware Horizon Client。
  2. 若安裝完成，開啟 VMware Horizon Client，點選畫面右上角 ，選擇「關於 VMware Horizon Client 」，即可看到版本資訊（組建編號為 8.X.X 以後，為 VMware Horizon Client 8）。
手冊下載

請開啟 VMware Horizon Client，點選「新增伺服器」，跳出小視窗後輸入伺服器名稱「vdiuag.cwb.gov.tw」，點選「連線」並依照指示輸入帳號、MOTP 密碼等。

手冊下載

如連線時出現憑證相關問題，請依下方步驟設定：

開啟 VMware Horizon Client，請點選右上方「設定（Setting）」，進入頁面後點選「安全性」，並於右方選擇「不要驗證伺服器身分識別憑證(Do not verify server identity certificate)」，設定後即可正常登入。

手冊下載

請聯繫資訊中心網管課劉子禎（#1293） 、鍾子健先生（#1293）協助處理。 手冊下載

  1. 請局內同仁協助至下方網址重置密碼或解鎖 https://itservice.cwb.gov.tw/account。
  2. 請聯繫資訊中心系統課林彥谷（#1398）、阮榮梃（#1284）或黃婉綺（#1288）協助處理。
手冊下載

VDI 浮動桌面：

  1. 請於 Horizon Client「設定」，檢查「VDI 浮動桌面」連線方式是否為「PCoIP」。
  2. 變更網路連線方式，試著改用其他 4G 或固網。
  3. 使用其他 Proxy Server 或停用 Proxy。
  4. 先登出 Horizon Client ，約 30 分鐘後再嘗試登入使用。

Horizon App：

  1. 請於 Horizon Client「設定」，檢查「應用程式」通訊協定是否為「PCoIP」。
  2. 變更網路連線方式，試著改用其他 4G 或固網。
  3. 使用其他 Proxy Server 或停用 Proxy。
  4. 先登出 Horizon Client ，約 30 分鐘後再嘗試登入使用。
如仍無法正常載入，請聯繫資訊中心系統課林彥谷（#1398）處理。
手冊下載

1.

請確認是否使用 EZ100 讀卡機。

2.

檢查 PC 或 NB 是否已安裝 EZ100 驅動程式及 HiCOS 卡片管理工具。

上述軟體若未安裝可能無法正常運作，請點以下官方連結下載：

► 讀卡機驅動程式 → https://reurl.cc/VD4e25
► HiCOS 卡片管理工具 → https://reurl.cc/b2lxdv
3.

前述步驟完成後，請至MOICA 內政部憑證管理中心點選 檢視憑證 IC 卡資訊 ，確認卡片資訊可正常讀出(如下圖)。

4.

開啟浮動桌面後，請確認「 USB 裝置」下 EZ100 讀卡機狀態為「關閉」，即可正常使用。

若前面步驟皆完成，仍無法使用，請檢查：

  1. 自然人憑證卡上面的晶片是否磨損。
  2. 自然人憑證卡插入方向是否正確。
  3. 使用其他自然人憑證卡，測試能否讀取正常。
  4. 除卡片本身外，亦請檢查讀卡機是否故障(嘗試使用讀卡機讀取晶片提款卡登入網路銀行)。
  5. 檢查防毒軟體是否啟用 USB 裝置監控機制。
  6. 離線 10 分鐘後再嘗試登入，使用系統新配發的虛擬桌面環境再行測試。
手冊下載

請聯繫資訊中心系統課林彥谷先生(#1398)處理。 手冊下載

  1. 檢查目前上網是否正常或嘗試使用其他網路( 4G 或固網)。
  2. 離線 10 分鐘後再嘗試登入。
  3. 請嘗試將電腦重新開機或重新安裝 VMware Horizon Client。
  4. 請聯繫資訊中心系統課林彥谷先生(#1398)處理。
手冊下載

  1. 確認目標主機帳密是否輸入正確，登入帳號是否被鎖定。
  2. 確認目標主機是否允許 VDI 桌面網段連入。(10.6.98.0、10.6.99.0)
  3. 確認目標主機是否允許 RDS Server連入。(10.6.129.11、10.6.129.12)
手冊下載

  1. 請檢查PC或NB網路連線是否正常？網路是否發生瞬斷不穩情形。
  2. 如網路正常，請嘗試重新連線並觀察連線情形。
  3. 請聯繫資訊中心系統課林彥谷先生(#1398)處理。
手冊下載

1.

請於 Horizon Client，按「設定」，選擇「VDI 浮動桌面」，先將連線方式改成「VMware Blast」並進行連線。

2.

請於 Horizon Client 中，按「設定」，選擇「應用程式」，先將通訊協定改成「VMwareBlast」並進行連線。

3.

以上調整後連線會出現連線失敗訊息，此時再將「VDI 浮動桌面」及「應用程式」連線方式再改回「PCoIP」，連線後即可正常顯示。

說明 : 發現 VMware Horizon Client 雖已設定 PCoIP 協定，但底層可能仍透過 VMware Blast 連線，因此藉由此方式處理。

手冊下載

使用 Apple、Linux 作業系統連入 VDI ，無法使用自然人憑證登入公文系統及行政資訊服務網，建議改用 Windows 平台進行 VDI 相關操作。

手冊下載

1.

將滑鼠移至左下角「視窗」圖示 ，按下滑鼠右鍵選擇「執行」。

2.

根據 VMware Horizon Client 安裝的路徑貼上指令，並按「確定」。

C:\Program Files (x86)\VMware\VMware Horizon View Client\vmware-view.exe --serverURL="vdiuag.cwb.gov.tw"
或
C:\Program Files\VMware\VMware Horizon View Client\vmware-view.exe --serverURL="vdiuag.cwb.gov.tw"
3.

完成二道密碼驗證登入後，點選右上方「設定」。

4.

點選左方「VDI 浮動桌面」，於右邊區域將「自動連線至此桌面平台」關閉，下次連線時就不會直接登入浮動桌面。
