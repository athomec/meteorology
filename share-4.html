顯示遠端主機圖檔

使用者在遠端主機做好圖檔需要顯示，需要依循下列方式處理：

  1. 使用 Putty RDS 應用程式，連接安管外主機(如安管外 HPC)或透過跳板連接安管內主機(如安管內 HPC)
  2. 執行程式或產製圖檔，如多個圖檔可壓縮處理後方便傳送
  3. 以電子郵件方式直接將產品寄送到外部電子郵件信箱
    mutt -s “Mail Subject” MailAccount@gmail.com a ${ OneAttachFile } < MailContent
  4. 接下來您就可以在個人電腦收信後進行圖檔或資料處理

增加 VDI 系統使用網段

  1. VDI 浮動桌面使用 原 IP 網段為 10.6.99.0 ~ 253
  2. 因應授權數增加，新增網段 10.6.98.0 ~ 253
  3. 同仁需要因應：主機存取限制的設定需要加入新網段

讀卡機與憑證測試

電腦需要具有可使用的讀卡機且可讀取自然人憑證，請依下列步驟測試：

1.

安裝 HiCOS 卡片管理工具 (https://moica.net.gov.tw/download_1.html)

2.

安裝 HiCOS 後進行環境檢測

3.

如果測試成功，但使用公文系統仍有問題，請洽資訊中心並告知上述測試結果

4.

如果讀卡機與憑證測試失敗，請先安裝 / 更新讀卡機驅動程式，測試成功後方可繼續測試公文系統

讀卡機驅動程式更新方法如下：

1.

在「開始」的地方按滑鼠右鍵，選擇「電腦管理」

2.

左側選擇「裝置管理員」，展開後尋找「智慧卡讀卡機」

3.

在「智慧卡讀卡機」設備上按滑鼠右鍵，選擇「更新驅動程式」

4.

依據您使用的讀卡機是否附上 Windows 10 的驅動程式，決定選擇「自動搜尋驅動程式」或「瀏覽電腦上的驅動程式」選項

備註：更新讀卡機驅動程式後請關機後重新開機，並且再度進入此畫面檢查讀卡機驅動程式安裝狀態，如果仍然無法安裝成功，則該讀卡機將無法正常使用。

手冊下載

  1. MOTP 帳號：兩階段認證系統，需要申請使用權限
  2. 資訊中心 AD 帳號 VDI：VDI 認證使用帳號與密碼
  3. VDI：虛擬桌面系統，提供 VDI 浮動桌面與 RDS 應用程式兩個功能 VDI
  4. VDI 浮動桌面：全功能 Windows 桌面，可提供所有 Windows 程式功能
  5. RDS 應用程式：執行單一應用程式，執行效率高
手冊下載

  1. VDI 系統只能連網本局安管外各主機，Internet、安管內、DMZ 與其他區域無法連線
  2. 如果要連網安管內、DMZ與其他區域需要 申請網路連線
  3. 被連線主機如有存取權限控制需設定允許 VDI 的 IP 連入
  4. VDI 浮動桌面使用 IP 網段為 10.6.98.0 ~ 25310.6.98.0 ~ 253、10.6.99.0 ~ 253
  5. RDS 應用程式使用 IP 為：10.6.129.510.6.129.5、10.6.129.6
